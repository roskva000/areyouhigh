## 2024-05-18 - Dynamic ARIA Labels for Toggle Buttons
**Learning:** Static `aria-label` attributes on toggle buttons (e.g., "Toggle Global Chat") lack explicit context for screen reader users regarding the current state or the result of the action. Using a dynamic `aria-label` based on the component's state (e.g., "Open Global Chat" vs "Close Global Chat") provides clearer, actionable context.
**Action:** Always implement dynamic `aria-label` attributes for icon-only toggle buttons, reflecting the specific action that will occur when the button is pressed, rather than a generic "Toggle" description.
